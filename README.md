# TerapiaCapilar
Este projeto é um site institucional desenvolvido em HTML, CSS e JavaScript, criado como parte de um projeto de extensão acadêmica. O objetivo é oferecer uma presença digital inicial para uma pequena empreendedora do ramo de terapia capilar, apresentando seus serviços, identidade visual e informações de contato de forma clara, moderna e acessível.
O site utiliza boas práticas de estruturação e responsividade, garantindo uma navegação simples e eficiente.
